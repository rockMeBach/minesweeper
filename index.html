Basic Minesweeper Game

How it works?

  • It generates a board randomly via Vanilla JS, consisting of 80 valid cells and 20 mines
  • When clicked on any cell, it first checks how many mines exist around it and then greys it out and displays the number of mines around it on that very cell
  • If it turns out that the cell is a mine, it shows all the bombs in red, alerts game over and reloads the page
  • If it turns out that the cell is empty, it recursively looks for empty cells around the cell that the player just clicked and greys them out
  • If any of the cells around that empty cell is not empty, it does the usual(greys it out and shows the number of adjacent mines)
